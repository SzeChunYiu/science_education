# Science Education Project

This repository is organized around four main areas:

- `docs/` — planning, runbooks, subject documentation, reviews, and research
- `src/` — generation, layout, animation, and publishing tooling
- `data/` — reusable assets and other durable shared inputs
- `output/` — generated scripts, dossiers, episode plans, and render outputs

## Current Structure

- `docs/foundation/` — core design and system documents
- `docs/operations/` — live workflow runbooks for publishing and platform operations
- `docs/subjects/` — subject-specific curriculum and inventory docs
- `docs/reviews/` — expert reviews and technical assessment material
- `docs/research/` — tool and workflow research notes
- `docs/reference/` — writer briefs, scene references, and checklists
- `src/publishing/` — YouTube and TikTok browser/publishing helpers
- `data/assets/` — reusable shared asset libraries
- `output/physics/` — current generated physics content tree
