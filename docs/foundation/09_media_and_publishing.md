## Narration
AI voice narration with a stable brand voice.
Style: calm, intelligent, clear, story-like, not overdramatic.

Generate: full narration audio, sentence timestamps, alternate short-form pacing version.

## Visual Style
- equation cards, simple diagrams, text overlays
- subtle motion graphics, recurring layout
- controlled color palette
- historical portrait slides only where permitted
- **Avoid a chaotic "AI slop" look**

Video production should use the documented scene layer:

- asset inventory
- scene template kit
- pilot scene board
- style frames
- animatic
- final render

## Video Structure

**Short-form:** hook → mystery → one idea → one image → one takeaway → CTA

**Long-form:** opening question → historical setup → problem → step-by-step explanation → selected derivation → meaning/interpretation → references + next topic teaser

## Platform Publishing

| Platform | Purpose | Style |
|----------|---------|-------|
| X | Serialized episodes, authority building, audience testing | Frequent, short, consistent |
| TikTok | Discovery videos, broad reach | Visual clarity, quick opening, one metaphor |
| YouTube Shorts | Discovery, teaser content, gateway to long-form | Educational density |

## Automation Rules

### Hard Fail (must not publish)
- fewer than two credible sources
- unsupported historical origin claims
- citation mapping failure
- script length constraints broken
- disclosure status unknown
- asset generation failed
- too similar to recent outputs
- missing required metadata

### Soft Warnings (allow review)
- historical accounts differ across sources
- analogy fidelity imperfect but acceptable
- script accurate but not vivid enough
- tone too similar to recent uploads

### Versioning
Store versions for: prompts, dossiers, scripts, media assets, publication payloads.
