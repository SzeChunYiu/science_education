# Physics Subject Docs

This folder contains stable physics documentation that should survive across production sessions.

Use these files in this order:

1. `curriculum.md`
2. `asset-inventory.md`
3. `scene-design/`

## Contents

- `curriculum.md`
  - module ordering and curriculum structure
- `asset-inventory.md`
  - reusable visual asset needs across the physics catalog
- `scene-design/`
  - pilot scene boards that translate finished scripts into production-ready visual beats

## Production Rule

Scripts and media belong under `output/physics/`.

Stable planning documents, inventories, and reusable scene-design packages belong here under `docs/subjects/physics/`.
