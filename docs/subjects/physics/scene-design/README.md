# Physics Scene Design

This folder is the bridge between finished physics scripts and actual video production.

These files are not final rendered media. They are reusable scene boards that future sessions can use to build:

- style frames
- animatics
- layout specs
- final renders

## Current Pilot Boards

- `classical_mechanics_newtons_laws_scene_board.md`
  - pilot board for Galileo's ramp / constant-acceleration episode
- `electromagnetism_induction_scene_board.md`
  - pilot board for Faraday induction / Maxwell handoff
- `quantum_matrix_mechanics_scene_board.md`
  - pilot board for the uncertainty-principle episode

## How To Use These Boards

For a new episode:

1. pick the closest pilot board
2. reuse the same scene rhythm where possible
3. map in episode-specific assets and equations
4. create `3-5` approved style frames
5. build the animatic
6. only then move to full render
